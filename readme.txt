This plugin allow you to insert in your posts or pages years stats, tot nr. of post,tot nr. of comments and some average.
== Description ==

With this plugin you can publish years stats about your blog:

*Nr. Post 	
*Avg post character 	
*Tot post character 	
*Avg Comments 	
*Tot Comments


== Installation ==

1. Upload `Wpit Blog Stats` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create stats and insert the right shortcode in a page or post.
